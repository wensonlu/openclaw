# BMS AI Validator

B端管理系统 AI 自主验收平台

## 项目概述

实现 **AI 自主开发 + AI 自主验收** 的端到端闭环，通过四层验收体系 + ReAct 修正机制实现自循环智能研发。

## 核心架构

```
┌─────────────────────────────────────────────────────────────────┐
│                      任务调度层 (Temporal)                       │
├─────────────────────────────────────────────────────────────────┤
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐            │
│  │ 构建验收 │→│ 静态验收 │→│ 动态验收 │→│ 语义验收 │            │
│  └────┬────┘  └────┬────┘  └────┬────┘  └────┬────┘            │
│       │            │            │            │                 │
│       └────────────┴─────┬──────┴────────────┘                 │
│                          ↓                                      │
│                   ┌─────────────┐                               │
│                   │ ReAct 修正环 │                               │
│                   └─────────────┘                               │
├─────────────────────────────────────────────────────────────────┤
│                      执行层 (Playwright + LLM)                   │
└─────────────────────────────────────────────────────────────────┘
```

## 模块说明

| 模块 | 路径 | 说明 |
|------|------|------|
| `core` | src/core | 核心实体、控制器、配置 |
| `scheduler` | src/scheduler | Temporal 工作流编排 |
| `executor` | src/executor | 四层验收执行器 |
| `reactor` | src/reactor | ReAct 修正引擎 |
| `generator` | src/generator | 用例自动生成器 |

## 快速开始

```bash
# 编译项目
mvn clean install

# 运行测试
mvn test

# 启动服务
java -jar target/bms-ai-validator.jar
```

## API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/projects` | POST | 创建项目 |
| `/api/v1/features` | POST | 创建功能 |
| `/api/v1/tests/generate` | POST | 生成验收用例 |
| `/api/v1/tests/{id}/run` | POST | 执行验收 |

## 技术栈

- Spring Boot 3.2
- Temporal 1.22 (任务调度)
- Playwright 1.40 (自动化)
- PostgreSQL 15+
- LLM (OpenAI GPT-4 / 腾讯混元)
